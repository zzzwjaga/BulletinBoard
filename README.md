<h1>BulletinBoard - доска для объявлений</h1>

Данный проект - это веб-приложение для доски объявлений, которое предоставляет такие функции как создание, редактирование, удаление, просмотр и добавление объявлений товаров и услуг в избранное. 
<h2>Типы и возможности пользователей</h2>
<h3>Неавторизованный пользователь</h3>
<p>Это пользователи, не прошедшие регистрацию в приложении или вход. Данному типу пользователей предоставлены следующие возможности:</p>
<ul>
<li>
  <h4>Регистрация в приложении</h4>
  <p>Пользователь может пройти регистрацию в приложении для использования полного функциоанала приложения. Для создания учетной записи ему необходимо будет указать имя пользователя, адрес электронной почты и пароль с подтверждением.</p>
</li>
<li>
  <h4>Авторизация в приложении</h4>
  <p>Если пользователь уже совершил регистрацию, то для использования полного функционала приложения он может совершить вход, введя данные учетной записи(электронная почта и пароль).</p>
</li>
<li>
  <h4>Просмотр объявлений списком</h4>
  <p>Пользователь может просматривать объявления в виде списка с их краткими карточками, включащими фото, название и цену товара из объявления.</p>
</li>
<li>
  <h4>Фильтрация  и сортировка объявлений</h4>
  <p>При просмотре полного списка объявлений пользователь пользователь может выбрать нужные с помощью настроек фильтров по типу объявления(товар, услуга), по категории, по цене и населенному пункту, где объявление дано. 
    Также ему доступна сортировка по дате создания, популярности(кол-во добавлений в избранное) и цене. </p>
</li>
<li><h4>Просмотр подробной информации об объявлении</h4>
   <p>Пользователь может просмотреть подробную информацию об объявлениии, включающую фото, название, описание, цену товара или услуги, а также контакты создателя объявления.</p>
 </li>
</ul>
<h3>Авторизованный пользователь</h3>
<p>Это пользователи, прошедшие регистрацию в приложениию Данный тип имеет следующие возможности:</p>
<ul>
<li>
  <h4>Просмотр объявлений списком</h4>
  <p>Пользователь может просматривать объявления в виде списка их краткими карточками, включащими фото, название и цену товара из объявления.</p>
</li>
<li>
  <h4> Просмотр подробной информации об обяъвлении</h4>
  <p>Пользователь может просмотреть подробную информацию об объявлениии, включающую фото, название, описание, цену товара, а также контакты создателя объявления.</p>
 </li>
 <li>
   <h4>Cоздание обяъяления</h4>
   <p>Пользователь может создать обяъявление, добавив цену, фото, описание товара и свои контакты для связи.</p>
 </li>
<li>
   <h4>Редактирование объявления</h4>
  <p>Пользователь может изменить какие-либо данные в созданном им объявлении.</p>
 </li>
  <li>
   <h4>Удаление объявления</h4>
    <p>Пользователь может удалить созданные им объявления в случае, если товар купили или по иным причинам. </p>
 </li>
  <li>
    <h4>Добавление товаров в избранное</h4>
    <p>Пользователь может добавить товар в избранное для быстрого вызвращения к нему и его отложенной покупке.</p>
  </li>
  <li>
    <h4>Выход из приложения</h4>
    <p>Пользователь может выйти из приложения по завершению работы в нем.</p>
  </li>
</ul>
<h3>Модератор</h3>
<p>Данный тип пользователя обладает административными правами и имеет следущие возможности:</p>
<ul>
  <li>
    <h4>Удаление объявления</h4>
    <p>Модератор может удалить любое добавленное объявление, например, при подозрении на мошенничество.</p>
  </li>
  <li>
    <h4>Удаление пользователя</h4>
    <p>Модератор может удалить аккаунт пользователя, например при многократном подозрении на мошенничество.</p>
  </li>
  <li>
    <h4>Выход из приложения</h4>
    <p>Пользователь может выйти из приложения по завершению работы в нем.</p>
  </li>
</ul>
